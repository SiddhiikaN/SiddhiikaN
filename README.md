### Projects <br>

**[APILoadGenAI](https://github.com/SiddhiikaN/APILoadGenAI)** <br> 

A CLI load testing tool that stress-tests API endpoints using AI-generated chaos payloads. Sends concurrent requests, tracks latency and failure rates, and produces a post-run analysis report. Built to surface how APIs behave under realistic, unpredictable load.

**[API Contract Verifier](https://github.com/SiddhiikaN/API-Contract-Verifier)** <br>

Compares two versions of a JSON API response and flags what changed — added fields, removed fields, value changes, and type changes. Assigns a risk score (HIGH / MEDIUM / LOW / SAFE) based on the nature of the diff. Built to catch breaking changes before they reach consumers.

**[Inventory Management System](https://github.com/SiddhiikaN/InventoryManagementSystem)** <br>

A Java desktop application for managing product stock with full CRUD operations. Built with a layered architecture.
